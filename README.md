# Teacher Copilot: AI-Powered Feedback Engine

**Team LOOPERS** *Harish Kumar (25BAI11056) & Manthan Awaya (25BAI11050)*

## 📌 Project Overview
Teacher Copilot is an intelligent assistant designed to streamline the grading process for educators. It addresses the significant time burden teachers face when checking assignments and writing repetitive feedback. The system analyzes student responses and generates thoughtful feedback suggestions while keeping the teacher in total control of the final message.

## 🚀 Core Features
**Student Answer Input:** Seamlessly accepts text, code, or structured answers
**AI Feedback Generation:** Creates contextual, constructive feedback using advanced language models
**Common Mistake Detection:** Identifies recurring errors across the class for targeted intervention
**Teacher Dashboard:** Centralized view for managing assignments and tracking student progress

## 🛠️ Technical Architecture
* **Frontend:** Next.js (React) for optimal performance
* **AI Integration:** Vercel AI SDK for model orchestration and streaming
* **Database:** Firebase Cloud Firestore for structured data handling
* **Hosting:** Vercel.
